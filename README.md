## 💻 Shell Scripts Description

### 🔹 Script 1 — System Identity Report

Displays essential system information such as:

* Linux kernel version
* Current user
* System uptime
* Date and time
* OS license

**Concepts Used:** Variables, echo, command substitution

---

### 🔹 Script 2 — FOSS Package Inspector

Checks whether Python is installed on the system and displays:

* Installation status
* Python version
* Short description using case statement

**Concepts Used:** if-else, case statement, package checking

---

### 🔹 Script 3 — Disk and Permission Auditor

Analyzes important directories:

* Shows directory size
* Displays permissions, owner, and group

**Concepts Used:** for loop, ls, du, awk

---

### 🔹 Script 4 — Log File Analyzer

Reads a log file and:

* Counts occurrences of a keyword (default: "error")
* Works with user-provided file

**Concepts Used:** while loop, if condition, command-line arguments

---

### 🔹 Script 5 — Open Source Manifesto Generator

Generates a personalized open-source statement by:

* Taking user input
* Creating a paragraph
* Saving it to a text file

**Concepts Used:** read, string handling, file output

## ▶️ How to Run Scripts

### Step 1: Give execute permission

```bash
chmod +x script_name.sh
```

### Step 2: Run the script

```bash
./script_name.sh
```

### Example:

```bash
chmod +x script1_system_info.sh
./script1_system_info.sh
```
